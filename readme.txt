This plugin brings a testimonials slider block to the new WordPress Gutenberg editor.

== Description ==

Easily add a testimonials to your WordPress post or page via the new Gutenberg Editor.

- Add Member Description
- Add Member Name
- Add Member Image
- Add Member Position


== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for Team Member Block"
3. **Install** the "Team Member Block" plugin
4. **Activate** "Team Member Block" from your Plugins page
5. **Insert Block** on your Gutenberg Editor and select "Team Member Block"
